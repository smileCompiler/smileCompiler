### Product Story

I build digital products that do more than look finished. They feel responsive, intentional, and technically grounded from the first interaction to the most demanding production edge case.

My strongest work lives at the intersection of **web, mobile, interaction design, and applied visual technology**: scalable interfaces, real-time flows, 3D product experiences, camera-based features, and backend systems that quietly keep everything stable.

- **Web applications:** Next.js, React, TypeScript, component architecture, SSR, API integration, performance optimization
- **Mobile experiences:** cross-platform app flows, responsive UI systems, real-time states, smooth navigation, production release thinking
- **3D configurators:** product customization, interactive scenes, material/color switching, camera controls, WebGL-ready experiences
- **Computer vision:** OpenCV, object tracking, detection flows, camera input processing, visual automation concepts
- **Backend foundations:** Node.js, Python, FastAPI, PostgreSQL, MongoDB, REST, GraphQL, authentication, validation
- **Delivery systems:** Docker, CI/CD, GitHub Actions, testing strategy, maintainable deployment workflows
